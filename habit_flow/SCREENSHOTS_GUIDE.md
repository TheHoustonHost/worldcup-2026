## App Preview Video (Optional)

**Length:** 15-30 seconds
**Content:** Quick walkthrough of key features

### Storyboard:
1. **0-3s:** Home screen showing today's habits (quick pan)
2. **3-8s:** Create a new habit (fast forward through UI)
3. **8-15s:** AI Coach screen with message
4. **15-20s:** Stats dashboard
5. **20-30s:** Show streak celebration animation

## Store Listing Assets Checklist

### Required:
- [ ] App icon (1024x1024 for iOS, 512x512 for Android)
- [ ] At least 2 screenshots
- [ ] Short description (100 chars)
- [ ] Full description
- [ ] Privacy policy URL
- [ ] Support URL

### Recommended:
- [ ] 5-10 screenshots
- [ ] App preview video
- [ ] Screenshots for tablet (if applicable)
- [ ] Different screenshot sets for each device type

---

## Design Tips for Screenshots

1. **Show real data** - Use realistic habit names and numbers
2. **Highlight key features** - Focus on what makes your app special
3. **Use captions** - Brief text overlay on each screenshot
4. **Consistent branding** - Same colors, fonts, style across all
5. **Show value** - Display premium features hint at upsell
6. **Localize** - Consider screenshots for different languages
